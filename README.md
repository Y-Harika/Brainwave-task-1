# Brainwave Task 1

##  Project Overview

This project is a **web-based application** developed as part of Task 1 during the Brainwave Matrix Solutions internship. It includes an interactive frontend built with **HTML, CSS, and JavaScript** to demonstrate responsive UI behaviour and basic scripting logic.

##  Description

The Brainwave Task 1 project showcases frontend development skills by building a simple web interface with user interaction using core web technologies:

* HTML for structure
* CSS for styling
* JavaScript for interactive features  

This project emphasizes clean layout, dynamic updates, and client-side scripting.
